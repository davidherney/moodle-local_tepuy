# A Moodle plugin: local_tepuy

This plugin is used with _Tepuy template_ in order to enhance the basic functionalities.

Package tested in: moodle 4.5 and 5.1

## QUICK INSTALL

Download zip package, extract the tepuy folder and upload this folder into local/.
